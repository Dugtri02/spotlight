# Spotlight

Spotlight is a Discord bot that rotates roles to highlight members, with configurable scheduling, eligibility rules, and fallback roles.

## Features
- Role rotation with configurable intervals and max users
- Status-based selection (online/offline/streaming)
- Blacklist and fallback roles
- Force rotation with cooldowns
- Per-guild configuration management
